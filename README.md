# GoodiBaddi - HR Verification Platform

## Project Overview
GoodiBaddi is a MERN Stack Progressive Web App (PWA) designed for HR teams and companies to verify employee work history and professional behavior before hiring.

## Features
- JWT Authentication
- Employee Management
- Employee Search
- Employee Profile
- Feedback System
- Dashboard
- Protected Routes
- Responsive UI
- Toast Notifications

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
